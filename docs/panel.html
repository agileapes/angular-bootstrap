Panel

Panels help you contextually separate sections of your code which out to be presented in their own place.

Compartmentalizing your content in a visually effective manner is done thus:

Example

This is a sample panel This is the first attachment This is the second attachment 
<ui:panel type="{{panelType}}" title="My Panel">
    This is a sample panel.
</ui:panel>
<ui:panel-attachment>
    This is the first attachment
</ui:panel-attachment>
<ui:panel-attachment>
    This is the second attachment
</ui:panel-attachment>
