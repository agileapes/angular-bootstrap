Progress Bars

Progress bars help you visualize the progress of a certain, on-going task.

Example

<ui:progress-bar progress="{{options.progress}}" label="{{options.label}}" active="{{options.active}}" striped="{{options.striped}}" type="{{type}}"></ui:progress-bar>
Do note that marking a progress bar as active will also automatically make it striped, whether or not you specify the striped property to be true.
