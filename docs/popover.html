<div id="myPopover" ng-non-bindable title="...">
    <ui:form-container label-size="5" orientation="horizontal">
        <ui:form-input type="text" id="firstNameEdit" label="First name" ng-model="values.first"></ui:form-input>
        <ui:form-input type="text" id="lastNameEdit" label="Last name" ng-model="values.last"></ui:form-input>
    </ui:form-container>
    <div class="row">
        <div class="col-lg-12">
            <input type="button" value="Close" class="pull-right btn btn-primary" ng-click="close();" />
        </div>
    </div>
</div>
The ng-non-bindable directive must be set on the popover element if we require that a correct scoping be presented to directives within the container.
