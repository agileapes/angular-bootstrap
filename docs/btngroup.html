Button Groups

Button groups can be used to present buttons that are semantically related or provide the same set of actions, in the same group. This is particularly useful for declaring toolbars of actions.

Example

First Header First Second Third Fourth 
<ui:button-group>
    <ui:group-button glyph="move"></ui:group-button>
    <ui:group-button glyph="upload" kind="success"></ui:group-button>
    <ui:group-button glyph="bullhorn" kind="information" label="Notify" position="after"></ui:group-button>
    <ui:group-button glyph="remove-sign" kind="danger" label="Delete" position="before"></ui:group-button>
    <ui:group-button glyph="check" kind="warning" label="Accept" position="before"></ui:group-button>
    <ui:dropdown id="myDropdown" label="Hello" position="after">
        <ui:dropdown-header>First Header</ui:dropdown-header>
        <ui:dropdown-item href="javascript:alert(1);" glyph="save">First</ui:dropdown-item>
        <ui:dropdown-item href="javascript:alert(2);" glyph="asterisk">Second</ui:dropdown-item>
        <ui:dropdown-divider></ui:dropdown-divider>
        <ui:dropdown-item disabled="true" href="javascript:alert(3);">Third</ui:dropdown-item>
        <ui:dropdown-item href="javascript:alert(4);">Fourth</ui:dropdown-item>
    </ui:dropdown>
</ui:button-group>
