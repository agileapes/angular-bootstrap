Button

Buttons are essential for navigating through your application, as well as creating visual landmarks for user actions.

Example

Print Ignore

HTML

<ui:button id="button1" type="primary" size="small"><ui:icon glyph="print"></ui:icon> Print</ui:button>
<ui:button ng-controller="mainCtrl" action="clickIt" id="button2" type="warning" size="small">
                <ui:icon glyph="ban-circle"></ui:icon>Ignore</ui:button>

JavaScript

<script type="text/javascript">
    mainCtrl = function ($scope) {
        $scope.clickIt = function () {
            alert("Clicked from controller!");
        }
    };
    $("#button1").on('click', function (e) {
        alert("Clicked from trigger!");
    });
</script>
