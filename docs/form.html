Forms

For form inputs, not much added markup is required. Generally speaking, the underlying frameworks provide quite a set of tools for accomplishing this. Therefore, what we deliver here, is mostly for the sake of unification and providing a uniform interface for creating forms across different frameworks.

Form Input

A form input component is a component that offers a data entry point to the user. This is usually not something that involves a choice.

For entry points which involve the user choosing from some predetermined set of options, see form select

Text Input

Text inputs are used primarily for un-formatted input where the user is allowed to enter data without restrictions.

<ui:form-input type="text" id="sampleTextInput" orientation="{{options.orientation}}" label-size="{{options.labelSize}}" state="{{options.state}}"
            label="Text field" feedback="{{options.feedback}}" placeholder="Text input" ng-model="sampleTextInput"></ui:form-input>

Form Select

Form selects are components which offer a set of predetermined input options to the user so that they might choose one or more of the available options

Combobox

Comboboxes are traditional <select/> components and are very well-suited to choosing among many items only one of which is the targeted choice.
