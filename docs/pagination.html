Pagination

At times it becomes necessary to paginate content to break it into more understandable pieces.

This can be very easily achieved in Angular-Bootstrap via a single directive.

Example


        

        

            
HTML

            
<ui:pagination id="x1" first="1" last="20" current="7" navigation="true" show="6"></ui:pagination>
