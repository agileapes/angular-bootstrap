BootstrapUI Test Suites the whole lot

Here, a comprehensive overview of the testing and specifications provided for BootstrapUI is presented.

Each collection consists of a set of specifications written using Jasmine and are run inside the browser.

Basics

Utilities

All the utilities that are built to help make the awesomeness that is BootstrapUI work out. They are however not an essential part of the framework itself, and can be used and seen individually and outside the package.

Configuration

The tests in this specification will decide how a configuration should be presented, and what minimal configuration the system requires to be provided with.
