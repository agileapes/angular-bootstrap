BootstrapUI Test Suites the whole lot

Here, a comprehensive overview of the testing and specifications provided for BootstrapUI is presented.

Each collection consists of a set of specifications written using Jasmine and are run inside the browser.

Basics

41

Utilities

All the utilities that are built to help make the awesomeness that is BootstrapUI work out. They are however not an essential part of the framework itself, and can be used and seen individually and outside the package. These are essentially extensions to the standard ECMA5Script specification.

 18

Configuration

The tests in this specification will decide how a configuration should be presented, and what minimal configuration the system requires to be provided with. The configuration is accessed via bu$configuration and modified via bu$configurationProvider

Services

7

Template Cache

BootstrapUI relies on an internal implementation of template cache which will replace the original $templateCache service by AngularJS and is able to perform some post-processing on templates.

 16

Registry Factory

A registry factory accessible through bu$registryFactory is a factory that will provide named registry singletons helping with saving content in a structured manner.

 12

Name Service

The bu$name service provides convenience methods for deriving BootstrapUI- and AngularJS-specific names following the globally observed naming conventions.

 62

Directive Compiler Service

The bu$directiveCompiler service which is in charge of gathering information about new directives post-bootstrap and compiling the necessary DOM subtrees as they are registered.
